Import PDF Invoice

Upload a PDF invoice from your glass supplier. The system will extract line items, create products, and record the invoice automatically.

What happens:
  • PDF text & tables are parsed for item data
  • New products are created (or existing ones updated by part number)
  • Invoice is recorded in the invoice tracker
  • A wholesale shipment record is created
  • Stock quantities are updated
Import Products from Excel

Upload an Excel (.xlsx) file with your product catalog. Columns are automatically mapped to product fields.

Expected columns (flexible matching):
  • Name — product name
  • Category — glass type
  • Make — car manufacturer
  • Model — car model
  • Year Start / End
  • Part Number / SKU
  • Price — selling price
  • Cost — wholesale cost
  • Stock — quantity
  • Description
